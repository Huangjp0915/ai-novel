# 文风指纹

## 通用规则

1. **自然流畅**：语言自然，符合网络小说风格
2. **画面感强**：场景描写生动，有画面感
3. **节奏控制**：合理控制节奏，避免拖沓
4. **对话自然**：角色对话符合人物性格

## 去AI味规则

### AI标记词限频
以下词汇每3000字不超过1次：
- 仿佛
- 忽然
- 竟然
- 不禁
- 宛如
- 猛地

### 叙述原则
1. 叙述者不替读者下结论，只写动作和场景
2. 禁止分析报告式语言（如"核心动机"、"信息落差"等）
3. 同一意象渲染不超过两轮
4. 方法论术语不入正文

### 词汇疲劳控制
- 避免重复使用相同词汇
- 注意同义词替换
- 保持语言多样性

## 文风要求

- 避免过度使用修饰词
- 保持简洁有力
- 注意语言的节奏感
- 符合网络小说读者的阅读习惯
