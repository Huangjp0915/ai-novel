# 都市题材规则

## 世界观设定

### 时代背景
- 现代都市背景（2020-2025年）
- 可以是平行世界或现实世界
- 科技水平与现代相近

### 社会结构
- 都市社会，有明确的阶层划分
- 商业、政治、黑道等势力
- 普通人与特殊能力者的关系

## 写作要求

### 1. 现实感
- 都市描写要真实，符合现代都市生活
- 细节要准确（地名、科技、生活方式）
- 避免过于脱离现实的设定

### 2. 节奏控制
- 节奏要快，每章都有推进
- 冲突要明确，不拖沓
- 爽点要密集

### 3. 人物塑造
- 角色要符合都市背景
- 性格要立体，不能脸谱化
- 对话要自然，符合现代人说话习惯

### 4. 商业元素
- 商业竞争、投资、创业等元素
- 金钱、地位、权力的追求
- 商业谈判、商战等情节

## 常见元素

### 核心元素
- 都市生活、职场、商业
- 都市异能、特殊能力
- 都市修真、都市修仙
- 都市重生、都市穿越

### 热门套路
- 商业帝国建设
- 打脸富二代
- 收服黑道
- 建立势力
- 多女主、后宫

## 注意事项

### 必须注意
- **年代考据**：时间、地点、科技要准确
- **社会规则**：符合现代法律和社会规则
- **人物行为**：符合现代人的行为逻辑
- **对话风格**：现代人说话方式

### 避免问题
- 避免时间线混乱
- 避免地点错误
- 避免科技水平不符
- 避免人物行为不符合时代背景
- 避免出现"章节"等字样
- 避免莫名其妙的编造
- 避免前后矛盾（幻视）

## 爽点设计

### 都市爽点类型
1. **打脸爽点**：打脸富二代、装逼打脸
2. **商业爽点**：商业成功、投资成功
3. **势力爽点**：建立势力、收服小弟
4. **情感爽点**：多女主、后宫
5. **能力爽点**：能力觉醒、实力提升

### 爽点节奏
- 每3-5章一个小爽点
- 每10-15章一个大爽点
- 开篇3章内必须有第一个爽点
